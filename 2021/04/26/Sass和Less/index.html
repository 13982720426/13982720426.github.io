Sass和Less

Sass和Less都属于CSS预处理器，CSS 预处理器定义了一种新的语言，其基本思想是，用一种专门的编程语言，为 CSS 增加了一些编程的特性，如：变量、语句、函数、继承等概念。将 CSS 作为目标生成文件，然后开发者就只要使用这种语言进行CSS的编码工作。

官网地址： http://lesscss.org/

VSCode插件：Easy LESS

官网地址： https://sass-lang.com/

VSCode插件：Easy Sass
