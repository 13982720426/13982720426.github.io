为什么null是对象

null有时候会被当做一种对象类型，但是其实是语言本身的一个bug，即对null执行 typeof null 时返回字符串 'object'，实际上，null本身是基本类型

原理是这样的，不同的对象在底层都表示为二进制，在JavaScript中二进制前三位都为 0 的话会被判断为 object 类型，null 的二进制表示全是 0 ，所以执行 typeof 时返回'object'
